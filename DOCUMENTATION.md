## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- **Next.js 16.1.3** - إطار عمل React مع App Router
- **TypeScript 5** - لغة البرمجة
- **Tailwind CSS 4** - تصميم الواجهات
- **shadcn/ui** - مكتبة المكونات (New York style)
- **Lucide Icons** - الأيقونات
- **Framer Motion** - الرسوم المتحركة

### الواجهة الخلفية (Backend)
- **Next.js API Routes** - واجهات برمجة التطبيقات
- **Prisma ORM** - إدارة قاعدة البيانات
- **SQLite** - قاعدة البيانات
- **bcryptjs** - تشفير كلمات المرور

### الأدوات والخدمات
- **Bun** - وقت التشغيل ومدير الحزم
- **ESLint** - فحص جودة الكود

## 🔐 نظام الصلاحيات

### الأدوار

| الدور | الاسم العربي | الصلاحيات |
|-------|-------------|-----------|
| president | الرئيس | جميع الصلاحيات + تفويض الصلاحيات |
| general_secretary | الكاتب العام | إدارة الأخبار، الرسائل، المستندات |
| treasurer | أمين المال | المالية، التقارير |
| deputy | النائب | صلاحيات محدودة (حسب التفويض) |
| admin | مدير | صلاحيات إدارية محدودة |

### أنواع الصلاحيات
- `all` - جميع الصلاحيات
- `news` - إدارة الأخبار والنشر
- `members` - إدارة حسابات الأعضاء
- `documents` - إدارة المستندات والملفات
- `messages` - قراءة والرد على الرسائل
- `reports` - عرض وإنشاء التقارير
- `finance` - إدارة الشؤون المالية
- `settings` - تعديل إعدادات الموقع

## 📅 تنسيق التاريخ

### الصيغة العربية:
```
بتاريخ : سنة/شهر/يوم
مثال: بتاريخ : 2026/03/15
```

### الصيغة الفرنسية:
```
Date : jour/mois/année
Exemple: Date : 15/03/2026
```

### دوال التنسيق (dateUtils.ts):
- `formatDate()` - التاريخ مع اسم الشهر
- `formatDateShort()` - التاريخ الرقمي
- `formatDateFull()` - التاريخ مع اسم اليوم
- `formatDateTime()` - التاريخ مع الوقت
- `formatDateRelative()` - التاريخ النسبي (اليوم، أمس، إلخ)
